Libros de Referencia

HTML & CSS: Design and Build Websites

Jon Duckett

Wiley, 2011 (Versión en Español)

Una introducción visual y práctica al desarrollo web, perfecta para diseñadores que quieren aprender a codificar sus diseños. Cubre HTML5 y CSS3 con ejemplos claros y ejercicios prácticos.

Ver recurso

CSS Secrets: Better Solutions to Everyday Web Design Problems

Lea Verou

O'Reilly Media, 2015

Técnicas avanzadas de CSS que resuelven problemas comunes de diseño web. Incluye trucos y soluciones creativas para crear interfaces modernas y eficientes usando solo CSS.

Ver recurso

Eloquent JavaScript (Español)

Marijn Haverbeke

No Starch Press, 2018 (Traducción al español)

Una introducción moderna a la programación con JavaScript. Cubre desde conceptos básicos hasta temas avanzados como programación funcional, manejo del DOM y desarrollo de aplicaciones web.

Ver recurso

Learning React: Modern Patterns for Developing React Apps

Alex Banks, Eve Porcello

O'Reilly Media, 2020 (2nd Edition)

Guía completa para aprender React desde cero hasta patrones avanzados. Incluye Hooks, Context API, testing y mejores prácticas para el desarrollo de aplicaciones modernas con React.

Ver recurso

Python Crash Course (2nd Edition)

Eric Matthes

December 2022, 552 pp.

Es uno de los libros más populares para aprender Python desde cero hasta proyectos reales, incluyendo un proyecto web completo con Django en la segunda parte.

Ver recurso

Python Web Development with Flask

Miguel Grinberg

2014 by Miguel Grinberg

Una guía paso a paso sobre cómo desarrollar aplicaciones web modernas con Flask. El autor es uno de los referentes más conocidos de la comunidad Flask.

Ver recurso

Artificial Intelligence: A Modern Approach (Sistemas Inteligentes)

Stuart Russell & Peter Norvig

1996 al 2025 Pearson All rights reserved

Considerado el libro más completo y autorizado sobre inteligencia artificial. Ideal para fundamentos de sistemas inteligentes, algoritmos de búsqueda, razonamiento, agentes, etc.

Ver recurso

Recursos Web y Documentación

Documentación Oficial

  • MDN Web Docs (Español) ↗ Documentación completa de tecnologías web: HTML, CSS, JavaScript y APIs. La referencia más confiable para desarrolladores web.
  • TypeScript Handbook ↗ Documentación oficial de TypeScript. Guía completa desde conceptos básicos hasta características avanzadas del lenguaje.
  • django ↗ Guía oficial del framework Django. Contiene desde la instalación hasta temas avanzados como middleware, señales, seguridad y despliegue.
  • Flask Documentation ↗ Documentación oficial de Flask, el microframework de Python. Incluye guías desde la instalación hasta el desarrollo de aplicaciones web completas.
  • aima Documentation ↗ Portal oficial del libro clásico de IA de Russell y Norvig. Contiene código de ejemplo, notebooks y materiales educativos sobre agentes, búsqueda heurística, redes bayesianas, entre otros.

Plataformas de Aprendizaje

  • W3Schools ↗ Tutoriales interactivos y referencias de programación web. Ideal para aprender HTML, CSS, JavaScript y muchas otras tecnologías web.
  • coursera ↗ Curso más popular para iniciarse en IA y aprendizaje automático. Explica conceptos clave como regresión, clustering, redes neuronales, y se apoya en ejercicios prácticos con Octave y Python.
  • realpython ↗ Portal con tutoriales, artículos y videos enfocados en temas prácticos de Python moderno. Ideal para mejorar habilidades con proyectos reales.
  • flaskpalletsprojects ↗ Herramienta de línea de comandos integrada en Flask para facilitar la ejecución del servidor, creación de proyectos y control de entornos. Ayuda a gestionar aplicaciones de manera profesional.

Herramientas de Desarrollo

  • Visual Studio Code ↗ Editor de código principal utilizado durante el curso. Potente, extensible y gratuito para desarrollo web.
  • React Documentation ↗ Documentación oficial de React. Incluye guías, API reference y tutoriales para el desarrollo con esta librería.
  • Python Documentation ↗ Documentación oficial IDE completo para Python, muy usado profesionalmente. Soporta debugging, testing, integración con bases de datos, entornos virtuales y autocompletado inteligente, facilitando el desarrollo ágil.
  • Flask Documentation ↗ Documentación oficial de comandos integrada en Flask para facilitar la ejecución del servidor, creación de proyectos y control de entornos. Ayuda a gestionar aplicaciones de manera profesional.
  • Django Documentation ↗ Documentación oficial de herramientas que amplían el framework Django. Incluye comandos útiles como runserver_plus, shell_plus, y diagramas de modelos. Muy útil para desarrolladores avanzados.

Nota sobre las Referencias

Todas las referencias bibliográficas han sido fundamentales en mi proceso de aprendizaje durante el curso de desarrollo web front-end. Los recursos están organizados según su relevancia y aplicación práctica en cada etapa del aprendizaje.

Los libros proporcionaron la base teórica sólida, mientras que la documentación web oficial fue esencial para resolver problemas específicos y mantenerse actualizado con las mejores prácticas de la industria.

Todos los enlaces han sido verificados y corresponden a recursos gratuitos o de acceso público utilizados durante el desarrollo del curso.
