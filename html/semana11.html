Cuaderno Semana 11 - Temas Aprendidos y Laboratorio

Temas Aprendidos

JSP/JAKARTA: Plataforma para construir aplicaciones web dinámicas en Java.

SPRING / SPRING DATA: Framework para simplificar desarrollo backend y acceso a datos.

Hibernate / JPA: Tecnologías para mapear objetos Java a bases de datos relacionales.

Se aprendió el procedimiento para crear aplicaciones empresariales robustas, usando APIs RESTful y gestión de datos.

Ejercicios de Laboratorio - Semana 11

Desarrollamos una API RESTful para la gestión de docentes con Jakarta EE y Spring. La aplicación permite operaciones CRUD y otras funcionalidades específicas:

  • Entidad Docente con atributos: idDocente, nomDocente, dirDocente, ciuDocente, emailDocente, fecNacimiento, tiempoServicio.
  • Operaciones CRUD (Crear, Leer, Actualizar, Eliminar).
  • Filtros por ciudad, años de servicio, edad promedio.
  • Validaciones: email, fecha de nacimiento, tiempo de servicio.
  • Documentación con Swagger y manejo global de excepciones.
  • Paginación en listado de docentes.

Resultados

La API fue probada con Postman, donde se comprobó la correcta gestión de docentes y las funcionalidades adicionales. Se lograron las validaciones y se implementó la paginación.

Reflexión Metacognitiva

¿Qué aprendí? Comprendí cómo construir APIs RESTful robustas con Jakarta EE y Spring, integrar validaciones, paginación y documentación, lo que facilita la gestión eficiente de datos en aplicaciones reales.

¿Cómo aprendí? A través de la práctica directa desarrollando el ejercicio, analizando documentación oficial y testeando con Postman, consolidando los conceptos teóricos con la experiencia práctica.
