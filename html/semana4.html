3. Reflexión: ¿Qué aprendí? ¿Cómo lo aprendí?

En este laboratorio aprendí a crear interfaces web completas usando HTML, CSS y JavaScript puro. El ejercicio 1 me permitió practicar el uso del DOM, inputs animados y zona drag & drop. En el ejercicio 2 aprendí a sincronizar sliders y valores numéricos para controlar colores RGB. Finalmente, el ejercicio 3 me permitió aplicar diseño responsivo y trabajar con media queries para adaptar el diseño a distintos dispositivos. Aprendí todo esto mediante la práctica directa, el análisis de ejemplos y la depuración del código.
