Reflexión Final

Análisis metacognitivo del proceso de aprendizaje

Reflexión Metacognitiva del Curso

A lo largo de este curso, he experimentado un proceso de aprendizaje profundo que me ha permitido no solo adquirir nuevos conocimientos técnicos, sino también desarrollar habilidades metacognitivas fundamentales para mi crecimiento profesional como desarrollador front-end.

¿Qué aprendí?

En este curso aprendí a ver la web desde adentro. Ya no solo me fijo en cómo se ve una página, sino en cómo funciona todo por detrás: cómo viaja la información, cómo los navegadores interpretan lo que escribimos y cómo se conecta todo a través de servidores y dominios. Entender eso me ayudó a construir páginas más seguras y bien hechas.

En la parte técnica, empecé con HTML y CSS, y luego aprendí a crear diseños que se adaptan a cualquier pantalla usando Flexbox y Grid. Después entré al mundo de JavaScript y TypeScript, donde ya podía hacer cosas interactivas: manejar eventos, crear animaciones, trabajar con el DOM… lo que hace que una web no solo se vea bien, sino que funcione bien.

Pero lo que realmente unió todo fue React. Ahí entendí cómo dividir una app en partes (componentes), cómo manejar estados y cómo hacer que todo responda según lo que pasa en pantalla. Aprendí a usar Hooks como useState, useEffect, entre otros, y también usé Tailwind para hacer diseños más rápidos y ordenados. Ahora siento que tengo las herramientas para crear aplicaciones web modernas, organizadas y funcionales.

¿Cómo aprendí?

Aprendí haciendo. Cada semana me enfrenté a retos que me obligaban a practicar, equivocarme, corregir y seguir mejorando. Las guías me guiaban, pero el aprendizaje real vino de ensuciarme las manos con código.

Trabajar con mis compañeros fue clave. Compartir ideas, ayudarnos con errores y pensar juntos cómo resolver un problema me dio una perspectiva más amplia.

Desde el inicio, la prueba diagnóstica me mostró en qué debía mejorar. Luego, con el enfoque de Aprendizaje Basado en Problemas, empecé a pensar más como desarrollador: no solo repetir, sino buscar soluciones reales. Y al final, más allá de la nota, la evaluación fue la mejor forma de ver cuánto había avanzado.

Semana a semana: mi evolución

Semana 1: Fundamentos web y herramientas

Conocí el ecosistema digital, configuré mi entorno con Visual Studio Code y aprendí que el orden al empezar es tan importante como el código en sí.

Semana 2: HTML, XML y CSS

Empecé a ver la estructura de la web. El manejo de Flexbox y Grid cambió mi manera de maquetar. Aprendí que el diseño también es lógica.

Semana 3: Bootstrap y Tailwind

Descubrí que no siempre hay que construir desde cero. Estas herramientas me ayudaron a hacer más con menos.

Semana 4: JavaScript y TypeScript

Fue una semana intensa. Aprender a programar desde la lógica pura hasta las animaciones fue un gran salto. Aquí reforcé mi pensamiento algorítmico.

Semana 5: React y Componentes

Empezó el enfoque moderno. JSX, props y estructura modular. Entendí que un buen componente puede hacer que todo el proyecto fluya mejor.

Semana 6: APIs y condiciones en React

Aprendí cómo se conectan las aplicaciones con servicios externos, y cómo manejar distintos escenarios desde la lógica del front-end.

Semana 7: Hooks

UseState, useEffect, useContext… Cada uno me dio más control sobre el comportamiento de mis aplicaciones. Fue el punto más avanzado, pero también el más revelador.
